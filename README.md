# Movie app project

## General

### Minimum OS version

- iOS: `iOS 10.0`
- Android: `Android 8.0 - Oreo`

## Setup

[React Native Development Setup](https://reactnative.dev/docs/environment-setup)

1. Install npm packages

```bash
yarn install
```

## Run the app

#### iOS

```bash
yarn ios
```

#### Android

```bash
yarn android
```
